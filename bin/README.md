# Piper Binaries

This repository no longer bundles or downloads Piper binaries.

Default runtime command is `python3 -m piper` (fallback: `python -m piper`).
Install Python and the Piper module locally, or pass an explicit
`piperBinaryPath` when creating `PiperTTS`.

Piper releases: https://github.com/OHF-Voice/piper1-gpl/releases
